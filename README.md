# 🤖 Chico — Seu Companheiro Financeiro

> Agente de IA desenvolvido voltado para orientação financeira personalizada com base em dados reais do usuário.

---

## 💡 Sobre o Projeto

O **Chico** é um agente financeiro inteligente criado para conversar com o usuário sobre sua vida financeira de forma simples, empática e eficiente. Ele analisa o perfil do investidor, o histórico de transações e os produtos financeiros disponíveis para oferecer orientações relevantes e personalizadas.

---

## 🧠 Capacidades do Chico

- Análise do **perfil de investidor** do usuário
- Consulta ao **histórico de transações** para identificar padrões
- Recomendação de **produtos financeiros** adequados ao perfil
- Atendimento empático com linguagem acessível
- Arquitetura baseada em **prompts estruturados** e base de conhecimento

---

## 🛠️ Tecnologias Utilizadas

- **Python** — lógica da aplicação
- **JSON / CSV** — estrutura de dados mockados
- **Engenharia de Prompts** — personalidade e comportamento do agente
- **IA Generativa** — motor de linguagem natural
